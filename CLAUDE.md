## Project Struture
This project is a express app design to be compatiable with sync
1. Controllers - Have functions that run on different routes
2. config - To setup thirparty api's and additional resources like a database
3. helpers - multipuprose functions
4. jobs - cron jobs
5. middlewares - middlewares of the project
6. routes - different api endpoints
7. misc - miscellaneous functions like seeding/testing scripts 
8. index.js - main startup file of the project
